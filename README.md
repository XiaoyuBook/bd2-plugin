# bd2-plugin

`bd2-plugin` 是一个面向 Yunzai-Bot 的《棕色尘埃2》查询插件。

插件目标：
- 为群聊和私聊提供 BD2 攻略/资料的快捷查询命令。
- 以 GameKee（`https://www.gamekee.com/zsca2/`）为主要资料来源进行信息整理与展示。
- 输出结构化、可读性高的查询结果，减少手动翻 Wiki 的成本。

计划覆盖的能力：
- 角色、装备、活动等基础信息查询。
- 常用索引与快速跳转。
- 统一的命令入口与错误提示，便于在云崽生态中扩展维护。

## 当前可用能力（v0.1）

- `#bd2 帮助` 或 `bd2 帮助`：查看命令说明。
- `#bd2 搜索 <角色名>` 或 `bd2 搜索 <角色名>`：按角色名查询，并返回该角色全部皮肤。
- `#bd2 角色 <角色名>` 或 `bd2 角色 <角色名>`：与搜索命令等价。
- `#bd2 测评 <角色名> [皮肤序号]` 或 `bd2 测评 <角色名> [皮肤序号]`：返回抽取建议、强度分析、环境分析。
- `#bd2更新`：拉取当前分支最新代码（仅 Bot 主人可执行）。

实现说明：
- 数据源为 GameKee 角色图鉴页（`/zsca2/jstj/122229?tab=jstj`）和角色评测页（`/zsca2/tj/{contentId}.html?tab=fzpc`）的 SSR 数据。
- 插件按角色名检索，不以皮肤名作为主检索入口。
- 命中单个角色时会一次性列出该角色所有皮肤（含基础技能摘要）。
- 测评默认返回图片卡片（角色图 + 皮肤强度 + 场景评级 + 三段分析），渲染失败时自动回退文字。
- 测评数据从结构化字段提取（`level / must_take / must_take_value / adviceValue / strenthValue / environmentValue`）。
- 执行 `#bd2更新` 后建议重启云崽以加载最新代码。
